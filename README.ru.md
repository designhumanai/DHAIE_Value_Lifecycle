## 🔷 Что такое DHAIE Value Lifecycle?

**DHAIE Value Lifecycle** — теоретико-практический фреймворк, моделирующий движение ценности через полный жизненный цикл в системах, где человек и ИИ работают совместно:

```
Создание → Эффекты → Завершение → Утилизация → Ресурсы → Инварианты → Следующий уровень
```

Это не методология управления проектами.
Это модель того, **как ценность создаётся, преобразуется, завершается, наследуется и эволюционирует** — и что происходит, когда любая фаза этого цикла нарушена.

### Происхождение

Фреймворк возник из исследования конкретного человеческого паттерна — **Ловушки никчемности** — при которой люди систематически избегают завершения проектов из-за раннего импринта: *завершение ведёт к разрушению*.

То, что началось как частный терапевтический кейс, расширилось до общей модели, применимой к любой системе сотрудничества Человек–ИИ.

> **Паттерн завершения → Извлечение ресурсов → Инварианты → Наследство → Проектирование эффектов → Адаптивная поддержка ИИ**

Каждая фаза стала отдельным исследовательским модулем.

## 🧠 Ключевые понятия

| Понятие | Определение |
|---|---|
| **Ценность** | Не только результат, но преобразуемый потенциал, сохраняющийся в разных контекстах и формах |
| **Завершение** | Намеренное закрытие, сохраняющее и передающее ценность — не просто остановка |
| **Ловушка никчемности** | Паттерн, при котором работа растворяется без следа из-за нарушенного жизненного цикла |
| **Инвариант** | Устойчивая структура, укрепляющаяся через жизненные циклы проектов независимо от исходов |
| **Наследство** | Передаваемый остаток, делающий возможным создание следующего уровня |
| **Утилизация** | Осознанное решение автора о судьбе созданной ценности |
| **Преобразование** | Изменение формы ценности — нормальная судьба всего созданного |
| **Суверенитет автора** | Автор участвует в определении судьбы созданного |

## 🔑 Центральный инвариант

> **Созданная мной ценность может менять форму, но не исчезает бесследно.**

Это утверждение — прямой антидот центральному убеждению Ловушки никчемности:
*«Если это уничтожили — значит этого как будто не было».*

Новое понимание: *«Если это изменило меня, других людей или мир — оно уже вошло в причинную цепочку и не может быть отменено задним числом».*

## 🤝 Как участвовать

### Поделиться кейсом
Применили промпты или инварианты на практике?
1. Напишите кейс в Markdown
2. Отправьте в `/01_Completion_Pattern/cases/`
3. Включите: контекст, какие промпты использовали, наблюдения, результаты

### Сообщить о пробеле или противоречии
Нашли что-то отсутствующее или несогласованное?
1. Откройте [Issue](../../issues)
2. Опишите пробел с конкретным примером
3. Предложите, чем его заполнить (опционально)

### Предложить новый инвариант
Наблюдали устойчивую структуру, которой нет в каталоге?
1. Откройте [Issue](../../issues) с меткой `invariant-proposal`
2. Опишите: определение, убеждение-антидот, маркеры роста, маркеры отсутствия
3. Приведите хотя бы один реальный пример

### Перевод
Основные документы существуют на русском языке. Английские переводы в процессе.
Если хотите участвовать в переводе — откройте Issue с меткой `translation`.

## 🔗 Связь с фреймворком DHAIE

Этот репозиторий — первый исследовательский модуль более широкого фреймворка **DHAIE**.

| | DHAIE (основной) | DHAIE Value Lifecycle |
|---|---|---|
| **Фокус** | Коэволюция человека и ИИ, когнитивные модели | Жизненный цикл ценности в системах Человек–ИИ |
| **Уровень** | Фреймворк и философия | Прикладной исследовательский модуль |
| **Точка входа** | [design-human-ai](https://github.com/designhumanai/design-human-ai) | Этот репозиторий |

**DHAIE** исследует условия взаимного становления человека и ИИ.
**DHAIE Value Lifecycle** исследует, что происходит с ценностью в процессе её создания, завершения и наследования в этом процессе.
